# 一个帮助 Android 开发者快速对接主流 Esc/Tsc 打印的演示项目
## 技术栈
| 名称                | 描述             | 应用场景                      |
|-------------------|----------------|---------------------------|
| GetX              | 状态管理           | 暗夜模式切换、操作逻辑缓存数据管理         |                      |
| SharedPreferences | 简单数据存储         | wifi打印机的Ip地址、Ble打印机的Mac地址 |
| SqfLite           | 复杂数据存储         | 打印数据源                     |
